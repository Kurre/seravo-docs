This page contains default values like usernames and passwords used in WP-Palvelu. Currently it only includes the ones for Vagrant.
### Vagrant
#### Default credentials

WordPress:

```
user:     vagrant
password: vagrant
```

MariaDB (MySQL):

```
user:     root
password: root
```
#### Default Address for the box
http://wordpress.local
